readme

This repository contains replication and extension code for the AQRD final project.

Due to data size restrictions, raw data (NIDS, CHIRPS, shapefiles) are not uploaded.
- NIDS: https://www.datafirst.uct.ac.za/dataportal/index.php/catalog/712/study-description
- CHIRPS: https://www.chc.ucsb.edu/data/chirps3

- shapefiles: https://dataportal-mdb-sa.opendata.arcgis.com/search?tags=2011
　Files used: `MDBDistrictMunicipalBoundary2011.gdb`, `MDBWard2011.gdb`

Scripts are organized in numerical order (01–03) . Scripts should be run in numerical order.
